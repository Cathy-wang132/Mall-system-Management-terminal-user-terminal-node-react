# Mall-system-Management-terminal-user-terminal-node-react
# node-电商商城平台项目源码（管理端+PC用户端）
### 1、介绍
本项目是基于现代前后端技术栈开发的电商商城平台，技术使用Node+Express框架+react + material-ui + ts+MySQL，分为管理端+PC用户端，项目结构清晰、技术先进，是学习实践电商系统开发的理想案例！
### 2、源码
下载地址：https://www.3woop.com/home/Index/projectInfo?goodsId=173&typeParam=1&subKey=0

页面有演示站，可查看预览~

开发不易，小偿提供源代码，谢谢理解~

### 3、使用技术
· 编程语言（后端）：Node.js 20.19.6

· 后端框架：Express

· 前端框架：React + TypeScript + Material UI

· 数据库：MySQL 8.0

· 包管理 / 构建工具：npm / pnpm（前端）

· 开发工具：VSCode

### 4、环境要求
· Node.js：20.19.6

· 数据库：MySQL 8.0

· 浏览器：Chrome / Edge（建议最新版）
